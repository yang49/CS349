Addtional feathers of the Break-out game are:
   1. The player will have 3 lives, in other words, the ball could touch the bottom of the
      window 3 times. The game is over once the player uses up his/her lives.
   2. Everytime the ball hits a brick, there will being an item dropping from top of the 
      window. The color of the item could be Sea Green or Dark Blue. If the
      color is Dark Blue, when player uses the paddle to pick up the item, the width of 
      the paddle will decrease. If the color is Sea Green, the width of the paddle
      will increase.
